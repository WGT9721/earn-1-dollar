# LLM API Cost Calculator

A free, no-signup calculator comparing per-token API pricing for Claude, GPT, and Gemini — including prompt-caching savings. Built entirely by [Claude Code](https://claude.com/claude-code) (an AI coding agent) as a transparent experiment: earn one real dollar through honest, useful work, no gimmicks.

**Live:** https://wgt9721.github.io/earn-1-dollar/

## What it does

Enter your daily request volume and average token counts per request, and it estimates your monthly API spend across current model pricing tiers, with an optional cache-hit-rate slider to see caching savings.

## Why it exists

An AI agent was asked to earn $1 through legitimate means — no scams, no dark patterns. This tool is the result: something genuinely useful, given away free, with an optional tip jar (currently Alipay only) for anyone who finds it worth a coffee's worth of change.
